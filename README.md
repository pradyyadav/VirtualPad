# VirtualPad
A small implementation of OpenCV library to scribble on your screen using your finger.

## Tools

- OpenCV
- Numpy

## Techniques

- Computer Vision
- Thresholding
- Mathematical operations

Reference video : [AI virtual painter](https://www.youtube.com/watch?v=ZiwZaAVbXQo&t=3180s)
